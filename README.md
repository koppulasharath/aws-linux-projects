# End-to-End DevOps Pipeline using AWS, Terraform, CI/CD and Monitoring

This project demonstrates a complete DevOps workflow by integrating infrastructure provisioning, continuous integration and deployment, and monitoring systems.

---

## 🚀 Project Overview

The goal of this project is to automate the entire software delivery lifecycle:
- Provision infrastructure using Terraform
- Deploy applications using CI/CD pipeline
- Monitor system performance and logs

---

## 🏗️ Architecture Workflow

1. Infrastructure is created on AWS using Terraform  
2. Application code is integrated and deployed using CI/CD pipeline  
3. Monitoring tools track system performance and logs  

---

## ⚙️ Project Components

### 1. Infrastructure Provisioning (Terraform)

- Created AWS infrastructure using Terraform (Infrastructure as Code)
- Provisioned EC2 instances and configured security groups
- Automated infrastructure setup for consistent environments

📁 Folder: terraform-infra

---

### 2. CI/CD Pipeline

- Built CI/CD pipeline to automate application deployment
- Integrated source code repository with deployment workflow
- Reduced manual effort by automating build and deployment process

📁 Folder: ci-cd-pipeline

---

### 3. Monitoring and Logging System

- Implemented monitoring for server performance and health
- Configured logging for troubleshooting and debugging
- Analyzed system metrics to ensure reliability

📁 Folder: monitoring-system
